  2. Task 2: Magic Letter

    First, read the description. There is an letter with a weird text. It's looks like base64 string.

    aHR0cHM6Ly9yZWRmaWxlcy5naXRodWIuaW8vCg==

    Let's decode it.

    I decoded that string with following command.

    echo "aHR0cHM6Ly9yZWRmaWxlcy5naXRodWIuaW8vCg==" | base64 -d

    I got a url, and checked there. I found a weird function in javascript code there. Looks like obfuscated.

    (function(){ var _0x4a65=["\x54\x48\x4D\x7B\x34\x6D\x34\x7A\x69\x6E\x67\x5F\x59\x30\x75\x5F\x46\x30\x75\x6E\x64\x5F\x54\x68\x33\x5F\x46\x69\x72\x35\x74\x5F\x46\x6C\x34\x67\x7D"]; })();

    So, I found a website DEOBFUSCATE JS to deobfuscate. I added console.log(_0x4a65) inside of the obfuscated function to read the variable. And got the first flag.

    (function(){ var _0x4a65=["\x54\x48\x4D\x7B\x34\x6D\x34\x7A\x69\x6E\x67\x5F\x59\x30\x75\x5F\x46\x30\x75\x6E\x64\x5F\x54\x68\x33\x5F\x46\x69\x72\x35\x74\x5F\x46\x6C\x34\x67\x7D"]; console.log(_0x4a65) })();
